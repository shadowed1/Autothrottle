# Automatic Low Power Mode switching with user-defined clockspeed limits for MacOS

- Designed for Apple Silicon
- Prevents Macbooks becoming too hot while enabling full performance when cool!
- Toggle between Low Power Mode without brightness changes.
- Customize clockspeed limit, cooldown duration, CPU usage threshold, and more!


# When opening the Autothrottle app, it will appear as a CPU icon in MacOS' Menu Bar.

# Click the CPU icon in the Menu Bar and click Autothrottle's icon to start/stop it.
